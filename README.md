## Agentic Development

```
  Architecte  ──────>  Agents IA  ──────>  Code, Tests, Deploy
     moi           orchestration          production demultipliee
```

Mon workflow quotidien repose sur l'orchestration d'agents IA pour faire avancer plusieurs fronts simultanement. J'utilise **Claude Code**, les **skills**, les **hooks**, et les **serveurs MCP** pour automatiser les taches repetitives et me concentrer sur la conception et l'architecture.

**Ce que ca change concretement :**
- Plusieurs features en parallele via des agents specialises
- Revue de code, refactoring et tests generes par IA
- Pipelines CI/CD concues avec assistance IA
- Plus de temps pour la reflexion architecturale et la securite

## DevSecOps

```
Code  -->  SonarQube  -->  Tests  -->  Build  -->  Deploy
 |            |              |           |           |
 v            v              v           v           v
Qualite    Securite     Fiabilite    Docker     Automatise
```

- **SonarQube** : Integration dans les pipelines pour l'analyse statique et la detection de vulnerabilites
- **Docker** : Containerisation et configuration d'environnements reproductibles
- **CI/CD** : Pipelines automatisees de bout en bout
- **Metriques** : Collecte et suivi de la qualite du code
